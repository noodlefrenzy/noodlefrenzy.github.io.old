    Welcome to Noodlefrenzy's GitHub Pages

    Hi there, my name is Mike Lanzetta but I typically go by @noodlefrenzy online. I work at Microsoft on a team that mostly develops open-source software in collaboration with startups or lab teams in enterprise organizations. Most of our team checks code directly into GitHub, and we maintain a fairly active presence online - blogging, tweeting, etc. We also give talks at conferences (some more than others), and we try and make sure problems and issues that people are having with Microsoft products (in particular Azure) get communicated (sometimes aggressively) back to the teams responsible.

    My GitHub Repos

    • node-amqp10: An AMQP 1.0 client implementation entirely in Node - no native code dependencies.
    • node-sbus and node-sbus-amqp10: node-sbus provides easy connection to Azure Event Hubs using any AMQP 1.0 client, tracking partition offsets in Azure. node-sbus-amqp10 uses my node-amqp10 as the implementation of the AMQP dependency (and is the one you should actively use in your project).
